# 班级投票助手

一个基于微信小程序云开发的班级投票助手应用，帮助班级快速发起和参与投票。

## 功能特性

- 🗳️ **发起投票** - 创建投票标题、描述、动态选项和截止时间
- 📋 **投票列表** - 展示所有投票，支持下拉刷新和上拉加载
- 👆 **参与投票** - 选择选项完成投票，支持防重复投票
- 📊 **结果展示** - 实时显示投票结果，包含票数、百分比和进度条

## 技术栈

- 微信小程序
- 云开发（CloudBase）
- 云数据库
- 云函数

## 项目结构

```
├── cloudfunctions/          # 云函数
│   ├── login/              # 获取用户openid
│   └── quickstartFunctions/
├── miniprogram/            # 小程序代码
│   ├── pages/
│   │   ├── index/         # 投票列表首页
│   │   ├── create/        # 发起投票页
│   │   └── detail/        # 投票详情页
│   ├── images/            # 图片资源
│   ├── app.js             # 入口文件
│   ├── app.json           # 全局配置
│   └── app.wxss           # 全局样式
└── project.config.json     # 项目配置
```

## 快速开始

### 1. 配置云开发环境

1. 在微信开发者工具中打开项目
2. 配置云开发环境ID（修改 `miniprogram/app.js`）
3. 创建数据库集合 `votes`
4. 部署 `login` 云函数

### 2. 数据库集合

创建 `votes` 集合，权限设置为「所有用户可读写」

### 3. 运行项目

```bash
# 在微信开发者工具中
# 点击「编译」运行项目
```
